# To-Do List на React + TypeScript

Простое приложение для управления задачами с сохранением в `localStorage`. Учебный проект для отработки React, TypeScript.

##  Функции
- Добавление задач
- Удаление задач
- Сохранение списка в `localStorage`

##  Технологии
- React (функциональные компоненты, хуки)
- TypeScript (типизация состояния и событий)
- CSS (адаптивная вёрстка)

##  Структура проекта
- `src/App.tsx` — главный компонент
- `src/App.css` — стили
